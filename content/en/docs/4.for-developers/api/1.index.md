# Misskey API

Misskey provides a public API.You can use this API to develop Misskey clients, integrated web services, bots, and more (hereinafter referred to as "applications").
A Streaming API is also available, allowing you to build applications with real-time capabilities.

:::tip

By using the official Misskey SDK or third-party libraries, you can simplify some of the procedures described in this document and make using the API even more convenient.

:::
