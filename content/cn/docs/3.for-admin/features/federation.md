# 联邦模式

Misskey 作为分布式平台，默认支持通过 ActivityPub 进行完全开放的联邦通信，但可以根据服务器的运营方针，可以选择对联邦范围进行限制或者完全禁用联合功能。

:::tip

服务器联邦功能默认已启用，因此通常（在使用联邦通信情况下）无需进行配置。

:::

## 设置方法

通过[控制面板 > 常规设置](x-mi-web://admin/settings)中的「联邦」进行设置。

### 全部

与所有服务器（不包括被屏蔽的服务器等）进行联邦通信。默认设置。

### 指定主机名

仅与特定服务器进行联邦通信（白名单）。在“允许联邦通信的服务器”中，输入允许联邦通信的服务器的域名，以换行分隔。

### 无

不与其他服务器进行联邦通信。部分联邦相关的设置画面或项目将被省略。
