# 反應

此功能可讓您透過添加表情符號，輕鬆傳達您對他人貼文的反應。
若要在 Misskey Web 中進行反應，請點擊貼文中的 + 圖示以顯示選擇器，然後選擇表情符號。
也可使用[自訂表情符號](./custom-emoji.md)作為反應。

## 自訂反應選擇器

在 Misskey Web 上，您可以依照自己的喜好，自訂選擇器中顯示的表情符號。
在設定的「表情符號調色盤」進行設定。

## 對於遠端貼文的反應

由於反應是 Misskey 的內建功能，除非遠端伺服器也是 Misskey，否則在大多數情況下，反應會以 Like Activity 的形式送出。一般來說，Like 通常會被實作為「最愛」。

## 來自遠端的反應

當 Misskey 從遠端接收到 Like Activity 時，會將其解釋為「❤」反應。

## 查看自己的反應清單

在 Misskey Web 上，您可以透過開啟個人資料頁面中的「反應」標籤，查看自己所做過的反應清單。
您也可以在隱私設定中將此反應清單設為公開。

## 接受表情反應

發布貼文時，你可以限制該貼文中可接受的反應類型。

- 全部：允許所有反應
- 全部（遠端僅限讚）：允許來自自己伺服器的所有反應，但來自其他伺服器的反應則會統一顯示為讚（❤）
- 僅限非敏感：允許所有反應，但來自自己伺服器的反應中，標記為「敏感」的自訂表情符號將無法使用
- 僅限非敏感（遠端僅限讚）：來自自己伺服器的反應中，標記為「敏感」的自訂表情符號將無法使用，而來自其他伺服器的反應則會統一顯示為讚（❤）
- 僅限讚：只能用讚（❤）來進行反應

請注意，此處指定的反應接受設定僅適用於自己的伺服器。例如，若將反應的接受範圍設為「僅限讚」的貼文在其他伺服器上查看，該伺服器上可能仍會顯示已附加的表情符號反應。
