# Misskey API

Misskey 提供了公开的 API。您可以利用此 API 开发 Misskey 客户端、与 Misskey 联动的 Web 服务、机器人 (Bot) 等（以下统称为“应用”）。
此外，我们还提供流式 (Streaming) API，方便您创建实时应用。

:::tip

通过使用 Misskey 官方 SDK 或第三方库 (Libraries)，可以简化本文档中说明的某些步骤，让 API 的使用更加便捷。

:::
