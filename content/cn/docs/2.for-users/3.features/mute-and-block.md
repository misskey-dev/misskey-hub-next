# 隐藏和屏蔽

如果您对某些用户不感兴趣，可以通过"隐藏"功能隐藏他们的动态。若需要更彻底的隔离，您也可以选择"屏蔽"该用户。屏蔽后，对方不仅无法查看您的内容，也无法与您进行任何互动。需要注意的是，隐藏操作不会被对方察觉，但屏蔽操作会明确告知对方已被屏蔽。具体选择哪种方式，请您根据自身情况决定。

:::tip

隐藏和屏蔽功能可以同时使用。

:::

:::warning

如果遇到违反使用条款的恶意用户，建议您考虑向平台管理员提交举报。

:::

在Misskey Web版中，您可以通过「设置 > 隐藏与屏蔽」查看您已隐藏或屏蔽的用户列表。

## 隐藏

将用户隐藏将阻止有关该用户的以下内容出现在 Misskey 上：

- 您的时间线和帖子搜索结果中的用户帖子（以及对这些帖子的回复和重新注释）
- 来自该用户的通知
- 消息历史记录列表中与该用户的消息历史记录
- 等等

要在 Misskey Web 上将用户隐藏，请打开该用户的用户页面上的菜单，然后按“隐藏”按钮。

:::tip

其他人不会收到您已将其隐藏的通知，也无法知道自己已被您隐藏。

:::

## 将转帖隐藏

如果某位用户大量转贴，您可以仅将该用户的转帖从时间线中隐藏。这不会影响该用户的普通帖子。

## 屏蔽

屏蔽用户后，该用户将无法看到您的内容，并且无法对您执行以下操作：

- 关注
- 添加到用户列表
- 回复或转发帖子
- 回应帖子或参与投票
- 私聊
- 等等

此外

- 如果用户在您屏蔽时已经关注您，该关注关系将被取消。
- 如果在您屏蔽某个用户时，该用户已将您添加到其用户列表中，您将从该列表中被移除。

要在 Misskey Web 上屏蔽用户，请打开该用户个人页面上的菜单，然后按下“屏蔽”按钮。

:::warning

对方不会收到您已屏蔽他们的通知，但由于他们将无法执行上述操作（例如关注您），因此可能会间接察觉到自己已被屏蔽。

:::

:::warning

对方将无法看到您的内容，但如果他们切换帐户或处于登出状态，仍然可以查看。请仅将此视为一种简单的辅助措施。

:::
