# 举报

在 Misskey 中，可以接收来自站内或站外用户对本服务器上内容的举报。

举报会出现在 [控制面板 > 举报](x-mi-web://admin/abuses) 中，**建议经常查看。**

## 处理举报

举报通常包含以下信息：

- 被举报的用户
- 举报内容
- 举报者



:::tip

有时会收到由远程服务器转发来的举报。在这种情况下，举报者可能会显示为该服务器的系统账号（`@instance.actor`）。

:::

如有需要，可以留下审批备注，用于记录举报相关的处理情况。该备注仅管理员及拥有监察员权限的账号可见。

处理完举报后，请将其标记为完成。

- 内容属实，并已采取某些措施时，选择“解决（确认）”
- 内容不属实，且未进行特别处理时，选择“解决（拒绝）”。

根据处理结果，可以区分“解决”的状态，并结束对应的处理。

## 转发到远程服务器

如果举报对象是远程用户，你可以将该举报转发到对方服务器。转发举报时，举报者会以匿名的系统账号（`@instance.actor`）的形式被转发。

## 举报通知

你可以通过 Webhook 或邮件通知接收举报。请打开 [控制面板 > 通报](x-mi-web://admin/abuses)，并在“通知设置”中进行配置。
