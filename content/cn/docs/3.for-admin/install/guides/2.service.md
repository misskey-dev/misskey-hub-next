# 使用第三方服务

您可以使用主机托管商提供的服务，轻松搭建 Misskey 服务器。这里我们介绍一些与 Misskey 兼容的服务。

:::warning

此处列出的服务是 Misskey 项目目前了解的部分服务，并非所有服务的完整列表。另外，<u>除非另有明确说明，请注意 Misskey 项目不会对这些服务进行功能验证，也不会作出官方认可或背书，无论服务提供商是否为 Misskey Project 的赞助方</u>**（如需咨询服务相关事项，请联系各服务的运营方，而非 Misskey Project）**。

:::

## 面向 Misskey 托管的专项服务

有些服务提供了专为 Misskey 定制的管理界面等功能。使用这些服务，即使您没有相关技术背景，也能够搭建 Misskey 服务器。

### XServer SNS

在 [XServer SNS](https://sns.xserver.ne.jp/) 中，Misskey 被提供为可使用的 SNS 软件之一。由于服务器的维护和版本更新均由 XServer 负责，几乎不需要任何技术知识，就能轻松搭建 Misskey 服务器。

## VPS 应用镜像

部分 VPS（Virtual Private Server）服务提供名为“简易安装”或“应用镜像”等功能，可让用户轻松完成环境搭建。其中也有一些默认支持 Misskey 的服务。这里将介绍其中的一部分服务。

### XServer VPS / Shin VPS

在 [XServer VPS](https://vps.xserver.ne.jp/) 和 [Shin VPS (シンVPS)](https://www.shin-vps.jp/) 中，Misskey 以应用镜像的形式提供。

详情请参阅：

- XServer: [使用 Misskey 应用镜像](https://vps.xserver.ne.jp/support/manual/man_server_app_use_misskey.php)
- Shin VPS: [使用 Misskey 应用镜像](https://www.shin-vps.jp/support/manual/man_server_app_use_misskey.php)



### Sakura VPS

在 [Sakura VPS](https://vps.sakura.ad.jp/) 中，Misskey 以启动脚本（Startup Script）的形式提供。

详细内容请参阅[“Sakura VPS Manual：启动脚本”](https://manual.sakura.ad.jp/vps/startupscript/startupscript.html)。
