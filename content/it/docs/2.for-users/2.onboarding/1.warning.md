# Precauzioni di base

Ci sono alcune cose da tenere a mente quando si utilizza il servizio Misskey.

- Poiché è decentralizzato, una volta che i dati caricati vengono cancellati, non è garantito che verrà eliminato da tutti gli altri server.(Questo può anche essere detto per Internet nel suo complesso.)
- Un post privato non garantisce che il server dell’altra parte lo tratterà come privato allo stesso modo.Attenzione quando si pubblicano informazioni personali e informazioni riservate.(Informazioni valide in generale, su Internet nel suo complesso)
- La funzionalità "Drive" di Misskey, non è un cloud storage.Serve soltanto per la gestione e il riuso dei file già caricati. **I file che vengono caricati, sono accessibili da chiunque ne conosca l'indirizzo URL, sia che siano stati collegati a una Nota, oppure no.** Ti preghiamo di usare altri servizi, per condividere file espliciti, oppure privati. Come ad esempio cloud storage presso terzi.
- Dopo aver terminato la registrazione, non è più possibile cambiare nome utente.Eliminando un profilo, il suo nome utente non sarà più disponibile a nessuno.
- La procedura di eliminazione di un account può richiedere molto tempo, poiché si tratta di una attività pesante.アップロードしたデータが多いケースではアカウントの削除が不可能なこともあります。
- サーバーによっては、広告収入により運営費を賄っていることがあります。また、アドブロッカーは広告とは全く関係のないコンテンツや機能を誤ってブロックすることがよくあり、クライアントの動作に支障をきたし、正常にMisskeyを使用できなくなるおそれがあります。そのため、Misskeyではアドブロッカーやそれに類する機能はオフにしてご利用ください。

これらをご理解いただいたうえで、楽しくサービスをご利用ください。
