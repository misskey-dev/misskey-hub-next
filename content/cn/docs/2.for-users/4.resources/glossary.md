# 术语表

与 Misskey 相关的术语表。

## Acct

用于表示特定用户的字符串，格式为`@username@host`。常用于提及用户等场景。

## ActivityPub

一种开放标准的协议（规范），用于构建被称为 Fediverse 的去中心化联邦式社交网络。遵循该协议，服务器之间可以相互连接，并在彼此之间交换内容、通知等信息。它是构建 Fediverse 网络不可或缺的协议。

## AiScript

一种可在 Misskey 中使用、运行在 JavaScript 之上的编程语言。详情请参见[此处](https://aiscript-dev.github.io/)。

## API

由 Misskey 服务器公开的接口，用于通过程序与 Misskey 进行交互。详情请参见[此处](../../for-developers/api)。

## Bot

由程序驱动运行的账号。

## CW

Contents Warning（内容警告）的缩写。一种可以在不进行操作的情况下不直接显示帖子内容的功能。主要用于隐藏较长的内容，或用于防止剧透等场景。详情请参见[此处](../features/note/#cw)。

## Fediverse

由 “Federated（Federation）”（联邦）与 “Universe”（宇宙）组合而成的造词。由实现了包括 Misskey 在内的各种去中心化软件的服务器所构成的、去中心化且联邦式的社交网络。去中心化软件还有 Mastodon、Pleroma、Pixelfed 等。

## GTL

全局时间线（Global Timeline）的缩写。关于时间线的详情可参考[此处](../features/timeline)。

## HTL

首页时间线（Home Timeline）的缩写。关于时间线的详情可参考[此处](../features/timeline)。

## LTL

本地时间线（Local Timeline）的缩写。关于时间线的详情可参考[此处](../features/timeline)。

## MFM

Markup language For Misskey 的缩写，一种可在 Misskey 中使用的标记语言。详情请参见[此处](../features/mfm)。

## Misskey Web

Misskey 官方的 Web 客户端（界面）。当通过 PC、智能手机等设备的 Web 浏览器访问 Misskey 服务器时，会自动显示该界面。

## NSFW

Not Safe For Work（工作场所不宜）的缩写。一种可将图片标记为“敏感内容”，并在未进行操作前不予显示的功能。

## 转发

指对已有帖子进行引用，或将该帖子作为新的帖子进行分享的行为，以及通过该行为生成的帖子。详情请参见[此处](../features/note/#转发)。

## STL

社交时间线（Social Timeline）的缩写。关于时间线的详情可参考[此处](../features/timeline)。

## 小蓝

是 Misskey 官方的吉祥物角色（看板娘）。详情请参见[此处](https://xn--931a.moe/)。

## 服务器

（服务器、实例）
向用户提供 Misskey 服务的实体。

## 自定义表情符号

由各个 Misskey 服务器的管理员或用户自行创建的表情符号。与之相对，非自定义的普通表情符号会被区分称为“Unicode 表情符号”。详情请参见[此处](../features/custom-emoji)。

## 控制面板

指 Misskey 服务器的设置界面。

## 系统账号

Misskey 服务器在与外部服务器进行交互等场景下使用的特殊账号。该账号内置于 Misskey 服务器中，并在需要时由服务器自动操作。

## 冻结

指账号被设置为不可用的状态。

## 网盘

用于在 Misskey 服务器上管理用户上传的图片、音乐等文件的功能。详情请参见[此处](../features/drive)。

## 帖子

指用户在 Misskey 服务器上发布的内容。可以包含图片、音乐等文件，以及投票等 Misskey 所定义的多种内容。详情请参见[此处](../features/note)。

## 代理账号

Misskey 服务器为获取外部服务器用户发布的内容而自动创建并操作的系统账号。当本服务器的用户在未关注外部服务器用户的情况下尝试获取其内容时，服务器有时会代为关注该用户。

## Misskist

指在 Misskey 服务器上拥有账号的用户。有时也用于特指其中的重度用户。

## 监察员

拥有服务器运营相关权限的用户，例如冻结垃圾账号、设置禁言、删除不当内容等。多数情况下，服务器管理员会在服务器内招募并任用监察员。

## 远程

指其他服务器，或隶属于其他服务器的对象。也可作为前缀使用，例如“远程用户”。与“本地”相对。

## 联合（联邦）

指多个服务器之间相互连接，彼此交换内容、通知等信息所形成的集合体。

## 本地

指自身服务器。也可作为前缀使用，例如“本地用户”“本地时间线”。与“远程”相对。
