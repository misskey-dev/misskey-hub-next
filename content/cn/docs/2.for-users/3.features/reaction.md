# 回应

这是一项可以通过添加表情符号，轻松对他人帖子作出回应的功能。
在 Misskey Web 中回应帖子时，请点击帖子上的 + 图标以显示选择器，然后选择一个表情符号。
回应中也可以使用[自定义表情符号](./custom-emoji.md)。

## 自定义回应选择器

在 Misskey Web 中，您可以根据自己的喜好自定义选择器中显示的表情符号。
该设置可在“设置” > “表情符号调色板”中进行。

## 远程帖子的回应

由于回应是 Misskey 的内置功能，除非远程服务器也是 Misskey，否则在大多数情况下，回应会以“点赞（Like）”的形式发送。在其他平台上，“点赞”通常会被实现为“收藏”功能。

## 来自远程的回应

当从远程服务器接收到“点赞”时，Misskey 会将其解读为“❤”回应。

## 查看自己的回应列表

在 Misskey Web 中，您可以通过打开个人主页中的“回应”标签页，查看自己发过的所有回应的列表。
您也可以在设置中，将此回应列表设为公开。

## 接受的回应类型

在发帖时，您可以限制该帖子所接受的回应类型。

- 全部：允许所有回应
- **全部（远程仅点赞）**：允许来自自己服务器的所有回应，但来自其他服务器的回应将仅以点赞（❤）的形式被接受
- **仅非敏感**：允许所有回应，但来自自己服务器的回应中，将无法使用被标记为“敏感”的自定义表情符号
- **仅非敏感（远程仅点赞）**：来自自己服务器的回应中，将无法使用被标记为“敏感”的自定义表情符号，且来自其他服务器的回应将仅以点赞（❤）的形式被接受
- **仅点赞**：无法使用点赞（❤）以外的方式进行回应

请注意，此处指定的回应接受规则仅在您自己的服务器上有效。例如，如果一篇帖子的回应接受规则被设置为“仅点赞”，那么在其他服务器上查看时，该帖子上可能仍然带有表情符号回应。
