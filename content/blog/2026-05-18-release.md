# NEW RELEASE: v2026.5.3

Misskey Projectは2026年5月18日、Misskey v2026.5.2, v2026.5.3 をリリースしました！

v2026.5.3 は v2026.5.2 が Docker 環境で起動しない問題を修正しています。

v2026.5.2 では、Unicode 17.0の絵文字への対応、RSA署名処理のネイティブ化などに加え、バグ修正やUIの改善などを行っています。

詳しくは[リリースノート](/docs/releases/)をご覧ください。
