MisskeyはAPIを公開しています。您可以利用 API 开发 Misskey 客户端、Misskey 链接的 Web 服务、机器人等（下简称“应用”）。
此外，我们还提供流式 API，方便您创建实时应用。

:::tip

通过使用官方的 Misskey SDK 或第三方库，您可以更方便地使用 API，例如简化本文档中描述的一些步骤。\\

:::
