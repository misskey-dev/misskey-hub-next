# 帖子

帖子是 Misskey 上发布的内容，包括文字、文件、投票等，是 Misskey 的核心概念。而创建帖子的操作则称为"发帖"。

:::tip

帖子这个名字来自英文单词 note，意为「简短的记录」。

:::

帖子发布后，会被加入[时间轴](./timeline)，您的[关注者](./follow)以及同一服务器上的其他用户都可以看到。

您可以对帖子做出[回应](./reaction)。当然，你也可以回复和引用。

将帖子加入[收藏](./favorite)后，您可以在之后轻松查看。

## 发布帖子

若要在 Misskey Web 上发帖，请按下屏幕上的铅笔图标按钮以打开发帖窗口。在发帖窗口中输入内容后，按下「发帖」按钮即可发布帖子。
您可以在帖子中附加图片、视频等任意文件，或添加[投票](./poll) 。此外，正文中可以使用 [MFM](./mfm) ，并可包含[提及](./mention)和[标签](./hashtag)。
另外，您还可以设置内容警告（CW）和公开范围等选项（详细内容将在后文说明）。

:::tip

在 Misskey Web 中，若您电脑剪贴板中已有图片，将其粘贴到发帖窗口的文本框即可添加该图片。

:::

:::tip

您亦可在发帖窗口的文本框内按<kbd class="key">Ctrl + Enter</kbd>来发布帖子。

:::

## 转发

引用现有帖子，或将其作为新帖子分享，以及由这些行为所创建的帖子，统称为转发。
当您想与您的追随者分享你最喜欢的某位用户的帖子时，或是想再次分享自己过去的帖子时，可以使用此功能。
您可以对同一条帖子进行不限次数的转发，但请注意避免过于频繁地使用，否则可能会对他人造成困扰。

:::warning

可见性设置为 「仅关注」或 「指定用户」的帖子无法被转发。

:::

若要在 Misskey Web 上删除转发，请在转发的时间显示旁边选择「...」，再选择「取消转发」。

## 内容警告（CW）

Content Warning 的缩写，是一种可在查看者未进行操作前隐藏帖子内容的功能。主要用于隐藏较长的内容，或用来防止剧透。
若要在 Misskey Web 上设置内容警告，请按下表单中的「隐藏内容」按钮（眼睛图示）。随后会出现一个新的文本框，您可以在其中填写内容摘要。

## 可见性

您可以为每条帖子设置其可见性。
若要在 Misskey Web 中设置可见性，请按下表单中发帖按钮左侧的地球图标。
可使用以下几种可见性设置：

### 公开

除了向所有人公开帖子外，帖子还会显示在该实例提供的所有时间轴（首页时间轴、本地时间轴、社交时间轴和全局时间轴）。

:::warning

当账户处于[禁言](./silence)状态时，无法使用此可见性。

:::

### 首页

帖子会向所有人公开，但不会出现在非关注者的本地时间线、社交时间线和全局时间线上。

### 关注者

帖子仅对自己的关注者公开，并会显示在关注者的所有时间线上。

### 指定用户

帖子仅对指定的用户公开，并会显示在指定用户的所有时间线上。

:::tip

v2025.7.0 之前（日语 UI），该选项称为「ダイレクト」。

:::

### “不参与联合”选项

如果启用此选项，帖子不会发布到联邦宇宙（远程服务器）。

## 置顶

将帖子置顶后，该帖子会始终显示在个人主页上。
要在 Misskey Web 中置顶帖子，请打开该帖子的菜单，然后选择“置顶”。

:::tip

个人主页上可以置顶多条帖子。

:::

## 新帖通知

当用户发布新帖子时，您可以收到通知。打开用户主页，点击关注按钮旁的详细按钮，然后点击“打开发帖通知”来启用此功能。
