# Page

In addition to notes, you can create pages on Misskey.In v13, pages are specialized for static content.

:::warning

Starting with v13, Misskey Play is used instead of Pages for creating dynamic pages.

:::

## Page Settings

Configure page-related settings.You can enter the following items.

- Title
- Page summary
- Page URL
- Center elements
- Font
- Hide Page title when pinned to profile
- Thumbnail

### Title

Enter the title of the page.It will appear in the list.

### Page summary

Enter a brief description.It will appear in the list.

### Page URL

You can customize the `●●●` part of `https://server-name/@user/pages/●●●`.

### Center elements

When enabled, the contents will be centered.

### Font

Choose between **serif** and **sans-serif**.

### Hide Page title when pinned to profile

Hides the page title when pinned.

### Thumbnail

You can set a thumbnail image using **Set thumbnail**.It can be displayed in lists and other views.

## Contents

Edit the page body.Pages can be generated by combining one or more of the following blocks.

- Section
- Text
- Image
- Embedded note

### Section

When a title is set, it is displayed as a heading.You can also place blocks inside.

### Text

Enter the text content.You can use MFM and custom emoji.

### Image

Display an image by selecting one from Drive.

### Embedded note

Enter a note ID to display the note.To embed a note from another server, enter its URL in the search and paste the displayed ID.
