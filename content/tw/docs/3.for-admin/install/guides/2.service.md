# 使用服務

Misskey 的伺服器可以透過主機代管服務等提供的功能，輕鬆完成建置。這裡將介紹部分支援 Misskey 的服務。

:::warning

此處所列的服務僅為 Misskey Project 目前了解的部分服務，並非所有服務的完整列表。此外，<u>除非另有特別說明，否則無論該服務的營運企業是否為 Misskey Project 的贊助商，皆未進行功能驗證或官方認證</u>**（有關服務本身的問題，請直接向各服務的營運方洽詢，而非 Misskey Project）**。

:::

## 專為 Misskey 託管打造的服務

有些服務提供了為 Misskey 專用而客製化的管理介面等功能。使用這些服務，即使沒有特別的技術知識，也能夠架設 Misskey 伺服器。

### XServer SNS

在 [XServer SNS](https://sns.xserver.ne.jp/) 中，Misskey 被提供為可使用的 SNS 軟體之一。由於伺服器的維護與版本更新皆由 XServer 負責，幾乎不需要任何技術知識，就能輕鬆架設 Misskey 伺服器。

## VPS 的應用映像檔

部分 VPS（Virtual Private Server）服務提供名為「快速安裝」或「應用映像檔」等功能，讓使用者能輕鬆完成環境建置。其中也有服務預設支援 Misskey。以下將介紹其中一部分服務。

### XServer VPS / Shin VPS

在 [XServer VPS](https://vps.xserver.ne.jp/) 與 [Shin VPS (シンVPS)](https://www.shin-vps.jp/) 中，Misskey 以應用映像檔的形式提供。

詳細資訊請參閱：

- XServer: [「使用 Misskey 應用映像檔」](https://vps.xserver.ne.jp/support/manual/man_server_app_use_misskey.php)
- Shin VPS: [「使用 Misskey 應用映像檔」](https://www.shin-vps.jp/support/manual/man_server_app_use_misskey.php)



### Sakura VPS

在 [Sakura VPS](https://vps.sakura.ad.jp/) 中，Misskey 以啟動腳本（Startup Script）的形式提供。

詳細內容請參閱[「Sakura VPS Manual：啟動腳本」](https://manual.sakura.ad.jp/vps/startupscript/startupscript.html)。
