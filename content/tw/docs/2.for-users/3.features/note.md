# 貼文

貼文是 Misskey 上發布的內容，包括文字、文件、票選活動等，是 Misskey 的核心概念。建立貼文本身的行為也稱為貼文。

:::tip

貼文這個名字來自英文單字 note，意思是「簡短的記錄」。

:::

建立貼文之後，貼文將被加入[時間軸](./timeline)，您的[追隨者](./follow)與同一伺服器上的使用者都可以看到。

您可以對貼文做出[反應](./reaction)，也可以回覆和引用。

將貼文加入[我的最愛](./favorite)後，您可以在之後輕鬆地再次查看。

## 建立貼文

若要在 Misskey Web 上建立貼文，請按下螢幕上的鉛筆按鈕以開啟建立表單。在表單中輸入內容後，按下「貼文」按鈕即可建立貼文。
您可以將圖像或影片等任何文件或[票選活動](./poll) 附加到貼文中。另外，文本中可以使用[MFM](./mfm)，還可以包含[提及](./mention)和[主題標籤](./hashtag)。
此外，您還可以配置內容警告和公開範圍等設定（詳細資訊將在後面解釋）。

:::tip

在 Misskey Web 中，若您的電腦剪貼簿中已有圖片，將其貼到表單的文字方塊即可附加該圖片。

:::

:::tip

在 Misskey Web 中，您也可以在文字方塊內按<kbd class="key">Ctrl + Enter</kbd>來發佈貼文。

:::

## 轉發

引用現有貼文，或將該貼文作為新貼文分享，以及由這些行為所產生的貼文，統稱為轉發。
當您想與您的追隨者分享您最喜歡的某個使用者的貼文時，或是想再次分享自己過去的貼文時，可以使用此功能。
您可以對同一則貼文進行不限次數的轉發，但請注意避免過於頻繁地使用，否則可能會對他人造成困擾。

:::warning

可見性設定為「追隨者」或「指定使用者」的貼文無法被轉發。

:::

若要在 Misskey Web 上刪除轉發，請在轉發的時間顯示旁邊選擇「...」，再選擇「取消轉發」。

## 內容警告（CW）

Content Warning 的縮寫，是一種可在未經查看者操作前隱藏貼文內容的功能。主要用於隱藏較長的內容，或用來防止劇透。
若要在 Misskey Web 上設定內容警告，請按下表單中的「隱藏內容」按鈕（眼睛圖示）。接著會出現新的輸入區域，您可以在其中填寫內容摘要。

## 可見性

您可以為每則貼文分別設定其可見性。
若要在 Misskey Web 中設定可見性，請按下表單中 「貼文 」按鈕左側的地球圖示。
可使用以下幾種可見性設定：

### 公開

除了向所有人公開貼文外，貼文還會顯示在該實例提供的所有時間軸（首頁時間軸、本地時間軸、社交時間軸和公開時間軸）。

:::warning

當帳戶處於[禁言](./silence)狀態時，無法使用此可見性。

:::

### 首頁

貼文會對所有人公開，但不會顯示在非追隨者的本地、社交或公開時間軸。

### 追隨者

僅向您的追隨者公開貼文，並顯示在其可見的時間軸中。

### 指定使用者

貼文僅對指定的使用者可見，並顯示在其可見的時間軸中。

:::tip

v2025.7.0 之前（日語 UI），該選項稱為「ダイレクト」。

:::

### 「停用聯邦」（僅限本地）選項

如果啟用此選項，則不會將貼文發布到聯邦宇宙（遠端）。

## 置頂

將貼文置頂後，該貼文會始終顯示在使用者頁面的頂端。
若要在 Misskey Web 中置頂貼文，請開啟貼文選項並選擇「置頂」。

:::tip

使用者頁面上可以置頂多個貼文。

:::

## 關注

當使用者發布新貼文時，您會收到通知。開啟使用者頁面，選擇追隨按鈕旁邊的詳細資料按鈕，然後按一下「開啟貼文通知」以啟用。
