# Misskey ID 卡

作为对支持 Misskey Project 的回馈，我们准备了实体的“Misskey ID 卡”！

**Misskey ID 卡是刻有序列号、专属于你的金属材质 ID 卡。**

我们在分量十足、具有厚重感的金属板上进行了精密加工，打造出了这款极具格调的金属卡。

<video src="/video/mi-id-card-teaser.mp4" muted autoplay loop></video>

其正面和背面均由 syuilo 精心设计。作为可以珍藏一生的物品，请务必将其留在身边细细把玩！

实物视频请看[此处](https://www.youtube.com/shorts/AdzzwxEa-WE) (由于光线原因可能看起来像金色，但实际为银色)

## 卡规格

- 尺寸: 54.0mm x 85.6mm
- 重量: 约 34g
- 材质: 1.0mm 厚镜面抛光不锈钢
- 加工：镜面加工（表面部分）、蚀刻加工（表面其他部分）、丝网印刷（Logo 部分）、激光雕刻（编号部分）
- 设计：由 syuilo 设计，附有序列号、条形码、QR码、小蓝签名刻印

作为可选项，也可选择在铭牌背面签上“syuilo”的名字。（免费）

## 发放对象

**将向累计赞助金额达到 2 万日元以上的支持者赠送一张。**

:::g-details{summary='2枚以上ご希望の場合'}

如果想要两张以上，每增加 3 万日元将额外赠送一张。

可额外领取的张数计算方式：从总赞助额减去 2 万日元后，再除以 3 万日元所得的数量。

例如，总赞助额合计为 5 万日元一共可获得 2 张，8 万日元以上一共 3 张，以此类推。

可以计算一共能领取多少张：

::X__Docs__mi-card__Calculator
::

此外，根据库存情况，可能无法完全满足您希望获得的张数。

※申请表单仅限提交一次。如需进行第二次及以后的申请，麻烦您通过“咨询表单”发送与申请表单相同的内容。

:::

使用美元（USD）进行的赞助，无论赞助时间为何时，统一按照 1 USD = 150 JPY 进行换算。其他货币的处理方式请与我们协商。

希望领取铭牌的用户，请在仔细阅读本页内容（包括[注意事项](#注意事项)）后进行申请。

另外，非常抱歉，由于配送方面的限制，**目前发放对象仅限居住在日本国内的用户。**

## 配送

配送将通过日本邮政进行。**免运费。**

- 发件时物流单号将通过邮件发送。
- 周末也可配送。
- 签收时可能需要核实本人身份。

卡片在经过检验后会进行仔细打包并寄出。如果在送达您手中时出现任何损坏，请向日本邮政进行咨询。

:::warning

卡片将配送至您在申请时填写的地址。
请注意，除申请者本人之外的地址无法进行配送。

此外，不支持指定具体的配送方式。

:::
