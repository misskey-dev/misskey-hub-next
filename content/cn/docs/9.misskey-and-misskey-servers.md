# Misskey 与 Misskey 服务器

“Misskey”与“Misskey 服务器”指的是不同的事物。

然而，由于名称相似，<b>加上“Misskey 服务器”经常被省略而简称为“Misskey”，因此非常容易引起混淆。</b>

实际上，我们也收到了许多将“Misskey 服务器”误认为是“Misskey”并来信咨询的情况。
（Misskey 并不是 Misskey 服务器的运营方。此外，本网站是“Misskey”软件的官方网站，并非“Misskey 服务器”的网站。）

请再次确认您是否将两者混淆了。

## “Misskey”

Misskey 是由志愿者社区“Misskey Project”开发的<b>“用于构建社交网络服务 (SNS) 的免费开源软件”</b>。

“开源”是指软件的源代码（设计图）是公开的。
“免费 (Free)”是指使用该源代码不会产生费用，任何人都可以自由使用。

有时 Misskey 会被介绍为一个“SNS”，但这是不准确的。

它归根结底只是“用于创建 Web 服务的软件”，而不是供您注册或登录的“Web 服务本身”。

:::tip

因此，作为服务提供方的“Misskey 运营团队”是不存在的。

如果您有过注册或登录的经历，那您登录的是 Misskey 服务器，而不是 Misskey 软件本身。

:::

:::warning

有时会有垃圾账号 (Spam) 自称为“Misskey 运营方”或“Misskey 客服”。

如前所述，此类组织或联系窗口根本不存在，请务必小心。

:::

由于任何人都可以自由使用 Misskey 的源代码，因此互联网上存在着由形形色色的人使用 Misskey 搭建并公开的 SNS（Misskey 服务器）。

此外，任何人都可以参与它的开发。

## “Misskey 服务器”

Misskey 服务器是指<b>“使用 Misskey 搭建的 SNS”</b>。

例如，“misskey.io”、“misskey.design”、“nijimiss.moe”等 SNS 都是 Misskey 服务器（Misskey 系服务器）。

另外，Misskey 服务器不仅仅由 Misskey 构成，它还结合了数据库软件“PostgreSQL”、“Redis”，以及 Web 服务器软件“Nginx”等多种软件组合而成。

截至 2025 年，已确认存在的 Misskey 服务器至少有 1,000 个以上。

这些服务器有些是由个人运营的，有些则是由企业运营的。

:::tip

不同的 Misskey 服务器，其运营者、运营方针以及服务条款都各不相同。

Misskey Project（Misskey 开发社区）并不负责运营 Misskey 服务器，也没有立场去管理、管辖或干涉各个 Misskey 服务器的运营。

至于“自定义表情符号”、“身份组（角色）”、“广告”、“禁用词列表（NG 词汇）”等，并非内置于 Misskey 中，而是由各个服务器的运营者自行设置的。

:::

如前所述，由于 Misskey（系）服务器很容易被简称为 Misskey，因此在网络上提及 Misskey 时，往往实际上是指某个特定的 Misskey（系）服务器。请注意这一点。

此外，为了表示“使用 Misskey 搭建”，Misskey 服务器上可能会显示“Misskey”的 Logo 或图标，或是带有“Powered by Misskey”等字样。

### “Misskey 系服务器”

使用由 Misskey 衍生（分支）出来的软件所搭建的服务器，有时会被统称为“Misskey 系服务器”。

有时，Misskey 系服务器也会被直接称为 Misskey 服务器。

因此，有些被当作“Misskey 服务器”使用的 SNS，实际上可能是“Misskey 系服务器”——也就是说，它是并未使用原版 Misskey 软件的 SNS。

## 示例

### 网站

打个比方，Misskey 就像是用作建立网站的软件“WordPress”。

而 Misskey 服务器则相当于使用“WordPress”建立起来的、一个个实际运作的网站。

### 公寓

Misskey 是“公寓的设计图”，而 Misskey 服务器则是“根据设计图实际建造出的一栋栋公寓”。

Misskey Project 是负责绘制建筑设计图的“建筑事务所”，而 Misskey 服务器的用户则相当于公寓的住户。

就像每栋公寓的房东（物业管理公司）都不一样，各个 Misskey 服务器的管理员和运营者也各不相同。

Misskey Project 仅负责公开设计图，并不会实际拿着设计图去盖公寓、不会去当公寓的房东或管理公司，也不会干涉已建成公寓的管理事务。

#### 更新

公寓的设计图会不断被修改和完善，有时也会增加新功能（例如新增室外楼梯）。

但设计图更新了，并不代表已经建好的公寓会自动跟着改变。

要让公寓符合最新的设计图，房东必须根据新版设计图重新对公寓进行改造（也就是系统更新）。

房东多久进行一次改造作业完全取决于房东自己，因此每栋公寓的“新旧程度”也有所不同。

#### 支持与客服

公寓内发生的邻里纠纷或设备故障等问题，只有房东能够处理。

由于 Misskey Project 只是建筑事务所，因此没有任何权限去干涉公寓内部的事务。
