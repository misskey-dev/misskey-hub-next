# 向项目捐赠

:::tip
如果您是企业并考虑赞助本项目，请查看[此处。](/docs/become-a-sponsor/)
:::

Misskey 并非商业项目，且完全免费使用，因此我们的唯一收入来源是大家的捐款。（虽然有些服务器可能会获得广告收入，但这属于该服务器运营者的收入，并不直接作为开发者的收入）

如果能得到您的捐助，我们将能够继续维持开发，这也是对本项目的重要贡献。我们目前通过以下方式接受捐款。（请在仔细阅读本页内容后再进行捐赠）

- 持续赞助
  - [Fanbox](https://syuilo.fanbox.cc/)
  - [Patreon](https://www.patreon.com/syuilo)
  - [GitHub Sponsors](https://github.com/sponsors/misskey-dev)
- 一次性赞助
  - （推荐）Kyash: @syuilo
  - [PayPal](https://paypal.me/syuilo)
  - [GitHub Sponsors](https://github.com/sponsors/misskey-dev)
  - 虚拟货币：`0xF1121108E21C92aAC7814137DD67e0d48eAC7Fdc`
  - [Amazon 心愿单](https://www.amazon.jp/hz/wishlist/ls/4JG4P6XKX9KD?ref_=wl_share)

:::tip

如果是大额捐赠（例如一次性捐款 3 万日元以上），通过 PayPal 等方式将会有高额手续费。请考虑使用 Kyash，又或者联系我来获取银行账户信息。

:::

:::tip

根据您的捐款金额，我们准备了以下回馈：

- 1000 日元以上：为您使用的 misskey.io 账号发放 mi 徽章
- 3000 日元以上：↑ + 在 Misskey 的致谢名单中署名
- 10000 日元以上：↑ + 在 Misskey 的致谢名单中署名及刊登自定义图标
- 20000 日元以上： ↑ + [颁发 Misskey ID 实体铭牌](/docs/mi-card/)

:::

:::warning

在任何情况下（包括因某种原因未能收到回馈等情况），捐款均不予退还。请在充分了解此规定的前提下进行捐赠。

:::

:::warning

**徽章发放以及在致谢名单中署名的操作并非自动进行。如需上述回馈，请附上能证明已捐款的截图或信息，并联系 `@syuilo@misskey.io`。**

（由于有时可能会漏看提及消息 (Mention)，如果未收到回复，请尝试多次提及。此外，从 misskey.io 以外的服务器发送的提及有时可能无法送达。如果确实无法取得联系，请通过 Misskey Project 咨询表单联系我们）

:::

此外，各服务器的运营者基本上也是没有任何收益的。维持服务器运行需要付出昂贵的成本，因此请您也考虑为您所在服务器的运营者提供赞助。

虽然这与软件开发没有直接关系，但正因为有了这些服务器，Misskey 项目才得以存在。因此，维持服务器运营与软件开发同样重要。
