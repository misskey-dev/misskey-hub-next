# 帖子

帖子是 Misskey 上发布的内容，包括文字、文件、投票等，是 Misskey 的核心概念。而创建帖子的操作则称为"发帖"。

:::tip

帖子这个名字来自英文单词 note，意为「简短的记录」。

:::

帖子发布后，会被加入[时间轴](./timeline)，您的[关注者](./follow)以及同一服务器上的其他用户都可以看到。

您可以对帖子做出[反应](./reaction)。当然，你也可以回复和引用。

将帖子加入[收藏](./favorite)后，您可以在之后轻松查看。

## 发布帖子

若要在 Misskey Web 上发帖，请按下屏幕上的铅笔图标按钮以打开发帖窗口。在发帖窗口中输入内容后，按下「发帖」按钮即可发布帖子。
您可以在帖子中附加图片、视频等任意文件，或添加[投票](./poll) 。此外，正文中可以使用 [MFM](./mfm) ，并可包含[提及](./mention)和[标签](./hashtag)。
另外，您还可以设置内容警告（CW）和公开范围等选项（详细内容将在后文说明）。

:::tip

在 Misskey Web 中，若您电脑剪贴板中已有图片，将其粘贴到发帖窗口的文本框即可添加该图片。

:::

:::tip

您亦可在发帖窗口的文本框内按<kbd class="key">Ctrl + Enter</kbd>来发布帖子。

:::

## 转发

引用现有帖子或将该帖子作为新帖子分享以及由此创建的帖子的行为称为转发。
当您想与您的追随者分享你最喜欢的某个使用者的帖子时，或者当您想要重新分享自己过去的帖子时，请使用此功能。
您可以无限次地重复相同的帖子，但请注意不要连续使用太多次，因为这可能会造成他人的困扰。

:::warning

可见性为 「仅关注」或 「指定用户」的帖子无法被转发。

:::

若要在 Misskey Web 上删除转发，请按转发时间显示旁边的「...」，然后选择「取消转发」。

## 內容警告（CW）

内容警告 Content Warning的缩写，是一种无需阅读者额外操作即可隐藏笔记内容的功能。它主要用于隐藏冗长的内容或防止剧透。
若要在 Misskey Web 上设定内容警告，请按下表单上的「隐藏内容」按钮（眼睛图示）。然后将出现一个新的文本框，您可以在其中写提示内容。

## 公开范围

对于每个帖子，您可以设定公开范围。
若要在 Misskey Web 中设定公开范围，请按下表单上 「帖子」按钮左侧的地球图示。
可以使用以下几种公开范围：

### 公开

除了向所有人公开帖子外，帖子还会显示在服务器上的所有时间轴（首页时间轴、本地时间轴、社交时间轴和公开时间轴）。

:::warning

当账户处于[禁言](./silence)状态时，无法使用此可见性。

:::

### 首页

笔记会向所有人公开，但不会出现在非关注者的本地时间线、社交时间线和全局时间线上。

### 关注者

笔记仅对自己的关注者公开。会发送到所有关注者的时间线上。

### 指定用户

笔记仅对指定的用户公开。会发送到指定用户的所有时间线上。

:::tip

在 v2025.7.0 版本之前，它被称为“私信”（ダイレクト）。

:::

### “仅本地”选项

启用此选项后，笔记将不再同步（联邦）到远程服务器。

## 置顶

将笔记置顶后，可以使其始终显示在您的个人主页上。
要在 Misskey Web 中置顶笔记，请打开该笔记的菜单，然后选择“置顶”。

:::tip

也可以同时置顶多条笔记。

:::

## 新帖通知

当用户发布新笔记时，您可以收到通知。请打开用户主页，点击关注按钮旁的详细按钮，然后点击“新帖通知”来启用此功能。
