# 小部件

小部件是可以放置在 Misskey 用户界面（UI）上，用于显示小型信息或进行操作的组件。

要编辑小部件，请切换到小部件编辑模式。切换方法因 UI 而异。
在小部件编辑模式下，您可以添加、删除、排序小部件，并对每个小部件进行设置。

### 个人资料

显示名称、用户名、头像和横幅图像。在使用多个账号时，可以更容易看清当前正在使用的是哪个用户。

### 服务器信息

显示服务器名称、域名、图标和横幅图像。

### 便利贴

可以保存文本内容。

### 时间线

显示时间线。可以切换显示主页、本地、社交、全局时间线。

### 日历

显示今天的年月日、星期，以及今天、本月、今年已过的百分比。包含在默认 UI 中。

### RSS 阅读器

以列表形式显示 RSS 订阅源的标题。

### RSS Ticker

依次滚动显示 RSS 订阅源的内容。

### 趋势

显示最近被频繁使用的话题标签。

### 时钟

用模拟时钟（指针式）显示当前时间。

### 活动

用点阵图显示活跃状况。

### 照片

显示最近网盘里的内容。

### 数字时钟

用数字时钟显示当前时间。

### UNIX 时钟

显示当前的 UNIX 时间戳。

### 联合

显示主要正在通信的其他服务器。

### 服务器球状列表

将其他服务器以球状显示。

### 发帖窗口

固定显示用于发布帖子的输入框。

### 幻灯片展示

依次显示网盘内特定文件夹的内容。

### 服务器指标

显示服务器的 CPU、内存、网络传输、存储容量。根据服务器环境情况的不同，有可能无法显示。

### 在线用户数

显示当前处于在线状态的用户数。

### 作业队列

显示与其他服务器收发帖子等内容的队列积压状态。

### 按钮

显示一个按钮。其动作逻辑通过 AiScript 编写。

### AiScript 控制台

显示可以执行 AiScript 代码的控制台。

### AiScript App

显示使用 AiScript 源码编写的应用程序。可以使用与 Misskey Play 相同的 UI。此功能于 v13 版本新增。

### 小蓝

显示看板娘“小蓝”。她的视线会随鼠标光标移动。

### 用户列表

显示列表。可以在 “更多！” -> “列表” 中设置列表。

### 点击器

显示一块曲奇饼干。可点击或轻触显示的曲奇。
