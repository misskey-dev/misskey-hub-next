# 页面

在 Misskey 上，您可以创建独立于帖子的页面。v13 以后专门用于静态页面。

:::warning

自 v13 起，需使用 Misskey Play 来创建动态页面。

:::

## 页面设置

用于设置与页面相关的项目。可填写以下内容：

- 标题
- 页面摘要
- 页面 URL
- 居中
- 字体
- 置顶时隐藏标题
- 封面图片

### 标题

输入此页面的标题。将显示在列表中。

### 页面摘要

输入简要说明。将显示在列表中。

### 页面 URL

可自行设置 `https://サーバー名/@ユーザー/pages/●●●` 中的 `●●●` 部分。

### 居中

启用后，内容将居中显示。

### 字体

可选择 **衬线字体** 或 **无衬线字体**。

### 置顶时隐藏标题

启用后，页面被置顶至个人资料时将隐藏标题。

### 封面图片

可设置 **封面图片**，用于列表等位置的显示。

## 内容

编辑页面本体。您可以组合以下一个或多个内容块来构建页面。

- 章节
- 文本
- 图片
- 嵌入的帖子

### 章节

可以设置专属的标题，并且可在其中增加额外的内容块。

### 文本

输入文本正文。可以使用 MFM 和自定义表情符号。

### 图片

显示图片。可使用网盘中的影像。

### 嵌入的帖子

输入帖子 ID 即可显示该帖子。若要嵌入其他服务器的帖子，请先在搜索中输入其 URL，再粘贴显示时获取的帖子 ID。
