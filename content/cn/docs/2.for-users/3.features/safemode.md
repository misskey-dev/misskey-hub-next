# 安全模式

在 v2025.8.0 及之后的 Misskey Web 中，如果因为插件、自定义 CSS 或主题导致客户端无法正常运行，系统提供了「安全模式」，可以在禁用这些功能的状态下启动。

可以通过以下方式进入安全模式：

- （启动时出现错误画面时）点击错误画面上的“以安全模式启动 Misskey”按钮
- （已连接键盘时）连续按下 `g` 键
- 在 URL 后添加 `?safemode=true`
- （使用 PWA 时）通过快捷方式选择 Safemode 来启动

在安全模式下，以下功能将受到限制。

- 无论设置为启用还是停用，所有插件都会被禁用，且无法进行新的安装。
- 自定义 CSS 将不会生效。
- 无论设置如何，都会以默认主题启动，且无法更换主题（仍可在主题管理页面中删除主题）。
- 页面上会始终显示“安全模式”的提示。

请善用安全模式，以找出并处理导致问题的部分。处理完成后，请在安全模式运行期间，从页面上始终显示的菜单中点击“关闭”，以停用安全模式。

:::tip

安全模式并非用于日常使用。处理完成后，请关闭安全模式，并以正常模式启动。

:::
