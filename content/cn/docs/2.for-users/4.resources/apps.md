# App

介绍一些用于使用 Misskey 的App应用（客户端）。

从浏览器登录 Misskey 服务器后，会显示时间线。时显示的就是 Misskey 唯一的官方客户端：**Misskey Web**。它是在浏览器中运行的应用，同时也支持 PWA，因此也可以像移动操作系统上的原生应用一样使用。详情请参阅[此处](/docs/for-users/stepped-guides/how-to-use-pwa/)。需要注意的是，Misskey 项目并未开发原生应用。

熟悉 Misskey Web 后，也可以试试由志愿者制作的非官方应用。这里将介绍广泛使用的应用（按字母顺序排列）。

- **Aria** (iOS, Mac, Android, Windows, Linux): 几乎可以使用 Misskey 的所有功能。UI 风格接近官方 Misskey Web 客户端。[官网](https://misskey.io/@aria_app)。[App Store](https://apps.apple.com/jp/app/aria-for-misskey/id6499410880)。[Google Play](https://play.google.com/store/apps/details?id=com.poppingmoon.aria)。[源码](https://github.com/poppingmoon/aria)。
- **Flare** (iOS, Mac, Android, Windows): 支持保存浏览历史，并提供 RSS 阅读器。支持 Mastodon、Bluesky、X，并可将它们的时间线整合为一个。[官网](https://flareapp.moe)。[TestFlight](https://testflight.apple.com/join/iYP7QZME)。[Google Play](https://play.google.com/store/apps/details?id=dev.dimension.flare)。[Microsoft Store](https://apps.microsoft.com/detail/9NLRN0BKZ357)。[源码](https://github.com/DimensionDev/Flare)。
- **Milktea** (Android): 可通过标签页快速访问时间线或特定的搜索结果。也可以为其他用户设置昵称。[Google Play](https://play.google.com/store/apps/details?id=jp.panta.misskeyandroidclient)。[源码](https://github.com/pantasystem/Milktea)。
- **Miria** (iOS, Mac, Android, Windows, Linux): 支持为标签页设置自定义表情，并提供 MFM 输入补全等为 Misskey 定制的独有功能。[官网](https://shiosyakeyakini.info/miria_web/index.html)。[App Store](https://apps.apple.com/jp/app/miria/id6449201469)。[Google Play](https://play.google.com/store/apps/details?id=info.shiosyakeyakini.miria)。[源码](https://github.com/shiosyakeyakini-info/miria)。
- **MissRirica** (iOS, Android): Misskey 的 iOS、Android 客户端。UI 几乎与 Misskey Web(v13) 一致，可接收推送通知。[源码](https://github.com/fruitriin/missRirica-client)。
- **Misscat** (iOS): 以在手机端提供更直观的操作体验为目标进行开发。[官网](https://yuiga.dev/misscat/)。[源码](https://github.com/YuigaWada/MissCat)。
- **SocialHub** (iOS): 付费应用，同时支持 Misskey、Mastodon、Bluesky、Slack、Tumblr。[官网](https://uakihir0.github.io/socialhub/)。[App Store](https://apps.apple.com/us/app/socialhub-socialmedia-client/id1474451582)。
- **ZonePane** (iOS, Android): 可将时间线、列表与频道以标签页形式管理；已读管理功能表现出色。是 Twitter 应用 TwitPane 的 Misskey 版，也支持 Mastodon 与 Bluesky。[App Store](https://apps.apple.com/app/id6747976082)。[Google Play](https://play.google.com/store/apps/details?id=com.zonepane)。
- **TootRain** (Mac): Mac 应用，可将时间线与列表中的帖子以评论弹幕式样实时显示在屏幕上。 [网站](https://b123400.net/tootrain/ja)。[源码](https://github.com/b123400/TootRain)。

不过，由于 Misskey 的功能更新十分频繁，对最新功能的适配也可能会有所滞后。若没有特别偏好，建议使用 Misskey Web。
