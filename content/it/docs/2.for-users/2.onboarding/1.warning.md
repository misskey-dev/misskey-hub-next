# Note importanti

Ci sono alcune cose da tenere a mente quando si utilizza un servizio Misskey.

- Poiché è decentralizzato, una volta che i dati caricati vengono cancellati, non è garantito che verranno eliminati da tutti gli altri server del fediverso.(Questo può anche essere detto per Internet nel suo complesso.)
- Un post privato non garantisce che il server dell’altra parte lo tratterà come privato allo stesso modo.Attenzione quando si pubblicano informazioni personali e informazioni riservate.(Informazioni valide in generale, su Internet nel suo complesso)
- La funzionalità "Drive" di Misskey, non è un cloud storage.Serve soltanto per la gestione e il riuso dei file già caricati. **I file che vengono caricati, sono accessibili da chiunque ne conosca l'indirizzo URL, sia che siano stati collegati a una Nota, oppure no.** Ti preghiamo di usare altri servizi, per condividere file espliciti, oppure privati. Come ad esempio cloud storage presso terzi.
- Dopo aver terminato la registrazione, non è più possibile cambiare nome utente.Eliminando un profilo, il suo nome utente non sarà più disponibile a nessuno.
- La procedura di eliminazione di un account può richiedere molto tempo, poiché si tratta di una attività pesante.Nei casi in cui siano stati caricati molti dati, potrebbe non essere possibile eliminare l'account.
- Alcuni server o istanze, potrebbero mostrare banner pubblicitari, per coprire le spese operative.Inoltre, le estensioni AdBlocker, spesso bloccano erroneamente contenuti e funzioni che sono completamente estranee alla pubblicità, interferendo con il comportamento del software. Chi li mantenesse attivati, potrebbe non essere in grado di utilizzare Misskey normalmente.Pertanto, si prega di disattivare il blocco degli annunci o funzioni simili in Misskey. Come ad esempio, il blocco canvas/fingerprint.

Comprendendo queste limitazioni, potrete utilizzare il servizio felicemente.
