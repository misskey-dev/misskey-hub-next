# 禁用 LTL/STL/GTL

在 Misskey 中，可以分别禁用 LTL、STL 和 GTL。若要切换启用/禁用状态，请在服务器控制面板中进行设置。

由于 LTL 和 STL 可以让用户看到该服务器上所有人的帖子，因此对于新用户来说，这具有无需刻意寻找用户、容易发现感兴趣用户的优点。
但与此同时，也存在一些缺点，例如：导致用户不再积极使用关注功能、不当内容容易被看到、或者因像聊天室一样运作而产生“小圈子”氛围，反而导致新用户难以融入。
由于不同服务器的优缺点权衡情况不同，因此提供了将其禁用的选项。
如果您觉得弊大于利，请考虑禁用这些时间轴。

:::warning

执行禁用操作后，用户可能会感到困惑，并且短期内用户数量可能会减少。因此，建议在禁用前慎重考虑其影响，并预留一定的时间提前说明情况，让用户有时间整理关注关系。

:::

此外，管理员/监察员不受这些时间线禁用状态的影响，他们仍然可以继续使用这些时间线。
