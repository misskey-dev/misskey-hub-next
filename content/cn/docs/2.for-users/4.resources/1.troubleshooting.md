# 故障排除

:::tip

也请一并参考[常见问题（FAQ）](./faq.md)。

:::

遇到问题时，请先查看本页。若没有对应的条目，或按步骤操作后仍未解决，请联系服务器管理员，或[提交问题报告](../../about-misskey/#加入讨论)。

## 客户端无法启动

多数情况下，这是因为您使用的浏览器或操作系统版本过旧。请将浏览器和操作系统更新到最新版本后，再试一次。

这种情况较少见；如果更新后仍无法启动，可能是缓存导致的。请清除浏览器缓存后，再试一次。

## 页面无法加载

如果客户端可以启动，但出现无法加载页面的错误，请先检查网络是否正常，并确认服务器是否未宕机。

这种情况较少见，但也可能是缓存导致的。清除浏览器的缓存，然后重试。

如果问题仍然存在，可能是服务器端的问题，请联系服务器管理员。

## 客户端运行缓慢

请先确认网络运营商是否发生故障或有计划维护，网络连接速度与网络设置是否正常，以及服务器端是否存在故障或正在维护。确认无误后，请尝试或检查以下设置：

1. Misskey Web 的设置从菜单打开“设置”，进入“偏好设置”，然后尝试以下设置：

- 在“辅助功能”中启用“减少 UI 动画”
- 在“性能”中禁用“对话框使用模糊效果”
- 在“辅助功能”中启用“使用系统默认字体”
- 在“常规设置”中，将“表情符号的样式”设置为“原生”

请在确认以下设置可能会导致部分 Misskey 功能无法正常使用后，再进行尝试：

- 在“行为”中禁用“禁用页面脚本”（现已无此选项，文档已过时）
- 在“辅助功能”中禁用“启用 MFM 动画”

2. Web 浏览器设置

- 在 Web 浏览器的设置中启用**硬件加速**
- 检查 Web 浏览器的插件或关闭其功能
- 检查 Web 浏览器的其他设置

3. 其他

- 检查您的 PC 或智能手机的系统设置

## Misskey Web 部分 UI 显示异常（例如背景变成透明等）

当更新包含 UI 改动时，可能会受到主题缓存系统的影响，从而出现这类显示问题。可以在客户端设置中点击“清除缓存”来修复。

:::warning

这里指的是清除“客户端”缓存。请不要清除 “浏览器” 缓存。

:::

## 通知、天线等的闪烁一直不消失

闪烁表示仍有未读内容。通常如果闪烁一直不消失，代表前面依然有未读项残留。
如果你认为自己已经全部读完，但闪烁仍然持续（可能是故障），可以在设置中强制将所有内容标记为已读。

## 无法转帖

可见性为“仅限关注者”或“指定用户”的帖子无法进行转帖。

## 部分元素在 Misskey Web 的 UI 上无法显示

如果您使用了广告拦截器等浏览器插件，可能会导致此类问题。请检查这些设置，或禁用它们后再尝试。

## Misskey Web UI 上有未翻译的部分

通常情况下，这并不是故障，而是因为 Misskey Web 的 UI 翻译尚未完成。请耐心等待，翻译工作完成后会更新。

如果您对 Misskey 的翻译工作有兴趣，欢迎您[参与翻译](../../about-misskey/#翻译文本)。
