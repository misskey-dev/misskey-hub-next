# 在线状态

用户的**在线状态**表示该用户使用 Misskey 的状态，**在线状态**会在用户页面等处的用户图标角落以指示器的形式显示。

<table>
	<tbody><tr>
		<th>颜色</th>
		<th>状态</th>
		<th>说明</th>
	</tr>
	<tr>
		<td>🟢绿</td>
		<td>在线</td>
		<td>当前可能在使用 Misskey</td>
	</tr>
	<tr>
		<td>🟡黄</td>
		<td>活动</td>
		<td>暂时离开，但 Misskey 仍在使用状态</td>
	</tr>
	<tr>
		<td>🔴红</td>
		<td>离线</td>
		<td>当前可能未在使用 Misskey 的状态</td>
	</tr>
	<tr>
		<td>⚫灰</td>
		<td>未知</td>
		<td>状态被设为**隐藏**，或因远程用户等原因而无法确定的状态</td>
	</tr>
</tbody></table>

您的在线状态会自动更改，无法手动设置，但可以通过设置将您的在线状态设置为**隐藏**。

:::warning

部分功能（如用户推荐）可能会基于您的在线状态进行处理，因此将您的状态设置为隐藏可能会使其他用户更难找到您。

:::
