# 搭建 Misskey 服务器

如果你不愿意使用 Misskey 的公共服务器，可以选择自行搭建。服务器搭建的方法有以下几种，你可以选择用你喜欢的方式来搭建服务器，点击下面的链接以查看详细的方法。

:::danger

请注意，一旦你准备好的域名或主机名在 Misskey 服务器上投入使用过，请不要重建数据库！否则将会造成联邦宇宙的混乱。

:::

:::warning

如果您修改了 Misskey 的源代码（包含 Fork），根据许可证规定，您有义务公开修改内容。有关详细信息，请参阅 [此处](/docs/for-admin/install/resources/forking/)。

:::

## 在 Xserver VPS 上搭建服务器（仅限日本用户）

在 XServer SNS 中，只需申请服务，即可自动创建 Misskey 服务器。

服务器的管理和维护也由运营方负责，所以不需要复杂的知识也能进行服务器运维。

详情请查阅[这里](https://sns.xserver.ne.jp/misskey.php)
